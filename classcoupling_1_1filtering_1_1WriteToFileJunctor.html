Detailed Description

template<unsigned int dim>
class coupling::filtering::WriteToFileJunctor< dim >

Combines two WriteToFile objects into one asymmetrical FilterJunctor. Used to output both primary and secondary cell data in an AsymmetricalFilterJunction.
