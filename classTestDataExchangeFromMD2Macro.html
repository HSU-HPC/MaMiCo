Detailed Description

template<unsigned int dim>
class TestDataExchangeFromMD2Macro< dim >

this class is used to handle test data exchange operations between dummy MaMiCo and a dummy macroscopic solver. For N processes, this class sends data from process n to n+1. The only data that are sent correspond to the points on the main "diagonal" of a field, that is cell data for cells at coordinates (n,n,n) (in global ordering).
