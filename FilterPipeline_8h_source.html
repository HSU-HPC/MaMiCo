31/*
32 * Manages different branches of filtering sequences.
33 * These filtering sequences may be interdependant by using another's sequences
34 * input or completely isolated. As this entire filtering process is applied
35 * during MD to Macro communication, it uses the MD simulation's output
36 * Macro-Cells as input and output. All configuration is made using an
37 * XML-config file and does not require recompilation when modified.
38 *
39 * @author Felix Maurer
40 */
