Detailed Description

template<class MacroscopicCell, unsigned int dim>
class coupling::sendrecv::FromMacro2MDSendOnly< MacroscopicCell, dim >

extends the generic SendReceiveBuffer for transfer of quantities from a macroscopic solver to the macroscopic cells of the coupling tool (incl. ghost cells). This version only sends data but expects not to receive any data, e.g. due to MD running on different ranks.

Philipp Neumann

sends information from the local macroscopic cells of a macroscopic solver to the macroscopic cells of MaMico (ghost cells are also allowed). Since the macroscopic solver can have an arbitrary distribution of cells on the processes, the buffer for sending the cell data is provided to this function in terms of an array of macroscopic cells including the respective global cell indices. Basically calls sendFromMacro2MDNonBlocking(...) and wait4SendFromMacro2MD(...) in a row.
