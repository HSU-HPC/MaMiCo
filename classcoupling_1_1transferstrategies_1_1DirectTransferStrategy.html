Detailed Description

template<class LinkedCell, unsigned int dim>
class coupling::transferstrategies::DirectTransferStrategy< LinkedCell, dim >

transfers and introduces mass and momentum directly into MD and to macroscopic solver. So, for example, if mass M is coming from the macroscopic solver, M is to be inserted into MD.

Author
Philipp Neumann
Template Parameters
LinkedCellthe LinkedCell class is given by the implementation of linked cells in the molecular dynamics simulation
dimrefers to the spacial dimension of the simulation, can be 1, 2, or 3
